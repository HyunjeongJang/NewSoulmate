# 3. 환승주인를 만드는 기술

- **서버**: JAVA 11, tomcat9, Servlet3, JDBC 
- **프론트엔드**:  JAVASCRIPT,JQUERY
- **데이터베이스**: ORACLE 11g
- **버전 관리**:MAVEN, Git with GitHub

<br>

# 4.  담당한 기능

- **회원**
    - 회원가입 / 회원탈퇴 / 로그인 / 로그아웃
    - 아이디 / 비밀번호 찾기 / 비밀번호 재설정
    - google 이메일 인증
- **후원**
    - 후원하기, 아임포트 결제연동
    - 개인 후원내역 , 후원 출금신청
- **사이트관리자**
    - 회원관리 회원리스트
    - 전체 후원내역
    - 보호소별 출금승인
    - 보호소 정보수정
